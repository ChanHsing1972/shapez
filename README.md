# Shapez

>《异形工厂》是一款关于建立工厂，自动创建以及关于形状组合的游戏。玩家在游戏中需要交付符合要求的越来越复杂的形状，在游戏中不断进步，并解锁升级以加快你的工厂生产速度。游戏很轻松，只需建造工厂，布好设施，无需操作即能自动创造出各种各样的几何图形。让我们也来写一个类异形工厂游戏吧！

本项目是一个模仿[异形工厂 (Shapez)](https://shapez.io/) 的游戏，为南京大学 C++ 高级程序设计课程的期末项目，由尘心独立开发制作。游戏基于 C++17 和 Qt5 框架开发，旨在通过完整复现游戏原作的核心逻辑与界面，展示 C++ 面向对象设计、STL 运用以及 GUI 界面开发的综合能力。

## 项目简介

在游戏中，玩家需要在地图上放置传送带、开采器、切割机、垃圾桶四种设施，运送或制造不同矿物至交付中心，获得点数。

- **开采器（Miners）**：布置开采器从地图上的资源点提取原始图形。
- **传送带（Belts）**：构建传送带网络，将资源从开采器输送到加工点或基地。
- **垃圾桶（Trash）**：布置垃圾桶，从四个方向输入矿物并永久清除它们。
- **切割机（Cutters）**：将物品纵向切开并分别输出。如果只需要其中一半的物品，请使用垃圾桶清除另一半物品，否则切割机将停止工作。
- **商店升级**：随着基地接收到指定数量的货物，可在商店使用点数升级设备。

操作方式如下：

- 左键单击/拖动：放置设施
- 右键单击/拖动：删除设施
- 1/2/3/4：选择设施
- R：旋转设施
- Esc：退出游戏

注意事项：放置传送带时，为确保其能够正确实现自动转弯，请匀速拖动鼠标，使其始终处于格子中线附近。

## 项目结构

```text
├── src/            # 核心源代码，包含所有的 C++ 逻辑实现和头文件
│   ├── main.cpp    # 程序入口

│   └── ui_classes  # 各个界面的控制类

├── docs/           # 课程项目说明文档及评分细则
├── assets.qrc      # Qt 资源配置文件
└── CMakeLists.txt  # CMake 构建脚本
```

## Quick Start

本项目采用 CMake 构建系统，跨平台支持 Windows / macOS / Linux。

### 环境依赖

- 支持 C++17 的编译器
- CMake (>= 3.10)
- Qt5 开发环境

### 编译步骤

进入项目根目录，执行以下命令：

```bash
# 1. 创建并进入构建目录
mkdir build
cd build

# 2. 生成构建系统文件
cmake ..

# 3. 编译可执行程序
cmake --build .
```

编译成功后，将在 `build` 目录下生成名为 `Shapez`（Windows 下为 `Shapez.exe`）的可执行文件。在终端中执行 `./Shapez` 或双击，即可运行游戏。
